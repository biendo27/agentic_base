# CLAUDE.md

## Project
Flutter app generated by agentic_base with Cubit state management.

## Architecture
Clean Architecture: data → domain ← presentation
- DI: get_it + injectable (auto-scan)
- State: flutter_bloc/Cubit + freezed sealed states
- Routing: auto_route
- Network: Dio + interceptors

## Commands
- `flutter pub get` — install dependencies
- `dart run build_runner build --delete-conflicting-outputs` — code gen
- `flutter test` — run tests
- `dart analyze` — lint check
- `make gen` — shortcut for code generation

## Conventions
- snake_case file names
- Package imports only (no relative)
- Each file < 200 LOC
- Features in lib/features/<name>/{data,domain,presentation}

## Boundaries
- Never edit *.g.dart, *.freezed.dart, *.gr.dart, *.config.dart
- Never commit .env files or secrets
- Never modify pubspec.yaml directly — use `agentic_base add/remove`
