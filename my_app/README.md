# My App

A Flutter application generated by [agentic_base](https://pub.dev/packages/agentic_base).

## Getting Started

```bash
flutter pub get
dart run build_runner build --delete-conflicting-outputs
flutter run
```

## Architecture

Clean Architecture with Cubit state management.

```
lib/
├── app/          # App config, bootstrap, flavors
├── core/         # DI, network, theme, router, error handling
├── features/     # Feature modules (3-layer)
├── shared/       # Shared widgets and utilities
└── l10n/         # Localization
```

## Commands

| Command | Description |
|---------|-------------|
| `make gen` | Run code generation |
| `make test` | Run tests |
| `make lint` | Run analyzer |
| `make build` | Build APK |
| `make ci-check` | Full CI check locally |
